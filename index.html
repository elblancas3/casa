Conecta la base de datos 🔥

Para que tú y Rafa vean los mismos datos en tiempo real, necesitas una base de datos gratuita de Firebase. Solo se hace una vez.

1
Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google.
2
Toca Crear proyecto, ponle cualquier nombre (ej. casacontrol), desactiva Google Analytics y créalo.
3
En el menú izquierdo busca Realtime Database → Crear base de datos → elige cualquier región → modo prueba (test mode).
4
Copia la URL que aparece arriba, tiene este formato:
https://tu-proyecto-default-rtdb.firebaseio.com
5
Pégala abajo y toca Conectar. ¡Listo! Manda el archivo a Rafa y ambos verán lo mismo.
